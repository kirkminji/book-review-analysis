# 뉴스 바이럴 지수 기반 베스트셀러 예측 분석 결론

## 분석 개요

**목표**: 뉴스 바이럴 지수를 활용하여 경제/경영 도서 베스트셀러 판매를 예측할 수 있는가?

**데이터**:
- 기간: 2025년 1월 ~ 12월 (51주)
- 도서: 141권 (베스트셀러 차트인 경험 있는 도서)
- 카테고리: 10개 (거시경제, 주식투자, 부동산 등)
- 뉴스: 카테고리별 주간 바이럴 지수

---

| 파일 | 설명 |
|------|------|
| `prophet_walkforward_decay.ipynb` | Prophet 시계열 예측 (카테고리 단위) |

## 핵심 발견

### 1. 뉴스 바이럴 → 도서 판매 인과관계

| 분석 방법 | 결과 | 해석 |
|-----------|------|------|
| 상관분석 | r < 0.1 | **거의 무상관** |
| Prophet 방향성 | 50.4% | **랜덤 수준** |
| ML R² (책 단위) | -0.43 | **예측력 없음** |

**결론: 뉴스 바이럴 지수와 도서 판매 간 인과관계가 통계적으로 약함**

---

### 2. Prophet 시계열 예측 (카테고리 단위)

| 지표 | 값 | 해석 |
|------|-----|------|
| 상관계수 (r) | 0.855 | 높음 |
| R² | 0.732 | 73% 설명력 |
| MAE | 8.17 | 평균 8점 오차 |
| **방향성 정확도** | **50.4%** | **랜덤 수준** |

**해석**:
- 카테고리별 **평균 판매 수준**은 예측 가능 (r=0.86)
- **다음 주 상승/하락**은 예측 불가 (방향성 50%)
- Prophet이 baseline은 잡지만, 주간 변동은 포착 못함

---

### 3. ML 모델 비교 (책 단위)

| 분할 방식 | F1 | R² | 해석 |
|----------|-----|-----|------|
| 시계열 분할 | 0.41 | 0.21 | 실전 성능 |
| 랜덤 분할 | 0.64 | 0.35 | 과대평가 (데이터 누출) |
| **책 단위 분할** | **0.27** | **-0.43** | **새 책 예측 불가** |

**결론**: 학습한 책의 패턴은 기억하지만, 새로운 책에 대한 예측력은 없음

---

### 4. 추가 피처 효과

| 추가 피처 | Prophet 개선 | ML 개선 |
|-----------|-------------|---------|
| 경제지표 (kospi, usd_krw, brent_oil) | -0.6% | 없음 |
| Prophet 잔차 | - | 없음 |

**결론**: 경제지표, Prophet 잔차 추가해도 유의미한 개선 없음

---

### 5. 피처 중요도 (ML)

| 순위 | 피처 유형 | 중요도 | 해석 |
|------|-----------|--------|------|
| 1~7위 | **카테고리 비율** | 높음 | 책의 고유 특성 |
| 8~15위 | Prophet 예측 | 중간 | 보조적 역할 |
| 16~30위 | 바이럴 교차, 경제지표 | 낮음 | 거의 무의미 |

**결론**: "어떤 카테고리의 책인가"가 가장 중요한 예측 변수

## 최종 결론

### 연구 질문에 대한 답변

**Q: 뉴스 바이럴 지수로 베스트셀러를 예측할 수 있는가?**

**A: 제한적으로만 가능**

1. **카테고리 수준**: Pr로 대략적ophet으인 판매 규모는 예측 가능 (r=0.86)
2. **책 수준**: ML로 개별 책 판매 예측은 불가능 (R²<0)
3. **방향 예측**: 다음 주 상승/하락 예측은 불가능 (50%)

### 실무적 함의

| 용도 | 가능 여부 | 권장 방법 |
|------|----------|----------|
| 카테고리별 트렌드 모니터링 | ✓ | Prophet |
| 개별 책 판매량 예측 | ✗ | 불가 |
| 마케팅 타이밍 결정 | △ | 뉴스보다 다른 요인 고려 |
| 재고 계획 | △ | 과거 패턴 기반이 더 정확 |

### 한계점 및 향후 연구

1. **데이터 한계**
   - 실제 판매량 대신 순위 기반 점수 사용
   - 베스트셀러 TOP 20만 분석 (21위 이하 데이터 없음)

2. **누락 변수**
   - 저자 인지도, 출판사 마케팅, SNS 버즈, 리뷰 수
   - 이러한 변수들이 판매에 더 큰 영향을 미칠 가능성

3. **인과관계 방향**
   - 뉴스 → 판매? or 판매 → 뉴스?
   - 역인과 가능성 검토 필요
